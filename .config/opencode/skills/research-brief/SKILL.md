## When to use me

- The task needs framework/library/API behavior confirmation.
- The task has high-uncertainty tradeoffs that need external evidence.
- The user asks for comparison, deep research, or source-backed recommendations.

## Workflow

1. Restate the research question and success criteria.
2. Query Context7 first for official documentation and examples.
3. If gaps remain, use SearXNG to find primary sources.
4. Separate observations from hypotheses.
5. Synthesize a recommendation with explicit uncertainty.

## Output contract

- Question
- Findings
- Evidence and sources
- Risks and unknowns
- Recommendation
